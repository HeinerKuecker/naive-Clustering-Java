# naive-Clustering-Java
Small Java class to cluster objects

Kleine Java-Klasse zum Zusammenfassen (Bündeln, Gruppieren) von Objekten.

Zum Erkennen ob zwei Objekte zum gleichen Cluster gehören, muss ein Interface ClusterCriteria implementiert werden.

To recognize whether two objects belong to the same cluster, a ClusterCriteria interface must be implemented.

Eine automatische Erkennung des Kriteriums für gemeinsame Cluster erfolgt nicht.

The criterion for shared clusters is not automatically recognized.

Eine automatische Erkennung des Kriteriums für gemeinsame Cluster erfodert eine Mindestgröße der Datenmenge.

An automatic detection of the criterion for common clusters requires a minimum size of the data volume.

Wenn die Datenmenge nur zwei Objekte enthält, ist dies dann ein Cluster oder zwei?

If the dataset contains only two objects, is this a cluster or two?

Dieses Repositiory ist eine Eclipse projectund kann in Eclipse importiert werden.

This repository is an Eclipse project and can be imported into Eclipse.
